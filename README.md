# Boilerplate with React, Webpack & SASS.

- Clone the repo
- Run `npm i`
- Run `npm start`
- Open http://localhost:3333/


### Dependencies:

- react
- react-dom

### Dev dependencies:

- babel
- babel-core
- babel-loader
- babel-preset-es2015
- babel-preset-react
- css-loader
- node-sass
- sass-loader
- style-loader
- webpack
- webpack-dev-server
